# vMonitor Platform

## vMonitor Platform là gì?

vMonitor Platform mang tới một giải pháp toàn diện cho việc thu thập, phân tích và cảnh báo trên dữ liệu Metric và Log từ GreenNode, các Cloud khác hay môi trường on-premise. Những thông tin này giúp bạn hiểu rõ hơn về hoạt động của các server/application của bạn và chủ động xác định được các vấn đề mà server/application đang gặp phải.
