# Agent Base

## 1. Agent Base là gì?

**Agent Base** là nền tảng cho phép xây dựng và triển khai các AI Agent một cách linh hoạt trên hệ sinh thái AI Stack. Agent Base cung cấp hai hướng triển khai phù hợp với nhiều đối tượng người dùng khác nhau:

* **Agent Base tự code** – Dành cho developer/AI Engineer muốn tự lập trình và tuỳ biến agent theo logic nghiệp vụ riêng.
* **Agent Base 1-click (OpenClaw)** – Cho phép triển khai agent nhanh chóng chỉ với một cú click, không cần lập trình.
