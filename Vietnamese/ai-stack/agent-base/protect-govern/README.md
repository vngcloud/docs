# Protect & Govern

## Tổng quan

Phần **Protect & Govern** tập hợp các công cụ giúp bạn giám sát và kiểm soát cách các Agent và người dùng tiêu thụ tài nguyên AI:

| Tính năng | Mô tả |
|---|---|
| **Rate Limit** | Giới hạn số Request và Token theo chu kỳ, áp dụng theo Model hoặc API Key |
