# Tổng quan

GreenNode AI AgentBase là một tập hợp các dịch vụ được quản lý hoàn toàn, được thiết kế để triển khai và vận hành các AI agent một cách an toàn và hiệu quả trên quy mô lớn. Nền tảng này được thiết kế để cung cấp hạ tầng cấp doanh nghiệp, bảo mật và khả năng quan sát cần thiết để đưa các bản mẫu AI agent vào môi trường production thành công. Nó hoạt động như một nền tảng thống nhất cho nhiều AI Agent framework khác nhau, đơn giản hóa các quy trình triển khai phức tạp.

---

## Các Module Chính

Hiện tại, AI AgentBase bao gồm **bốn dịch vụ dạng module: Runtime, Access Control, Insight, Memory**. Mỗi module hoạt động độc lập hoặc phối hợp để cung cấp các chức năng thiết yếu cho hệ sinh thái AI agent của bạn.

| Module                   | Mô tả                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| ------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Runtime**        | Runtime là một môi trường thực thi serverless an toàn, triển khai và scale các AI agent mà không cần quản lý hạ tầng. Mỗi agent chạy trong một container cô lập, ngăn chặn rò rỉ dữ liệu giữa các phiên làm việc, đồng thời tự động xử lý việc scale cho các tác vụ đồng thời và chạy lâu. Runtime hỗ trợ bất kỳ agent framework nào, nhiều giao thức giao tiếp (HTTP, MCP, A2A).                                                                                                                                                                                                                                                                                                 |
| **Access Control** | Access Control được thiết kế để đảm bảo tương tác an toàn và được ủy quyền giữa các AI agent và các tài nguyên nội bộ hoặc bên ngoài. Module này tập trung quản lý credential, token và quyền truy cập, điều này rất quan trọng cho bảo mật và tuân thủ quy định của doanh nghiệp.                                                                                                                                                                                                                                                                                                                                                                                 |
| **Memory**         | Module Memory cung cấp các khả năng cho phép agent hiểu các tương tác đang diễn ra và duy trì tính liên tục theo thời gian. Nó loại bỏ nhu cầu các nhóm phải tự xây dựng và duy trì cơ sở dữ liệu Memory, hệ thống vector indexing hoặc các Memory pipeline của riêng mình. Memory ngắn hạn duy trì các luồng hội thoại trong phiên làm việc, cho phép agent hiểu các tham chiếu, theo dõi các cuộc đối thoại phức tạp và xây dựng các phản hồi mạch lạc mà người dùng không cần nhắc lại thông tin. Memory dài hạn lưu trữ thông tin thiết yếu qua các phiên làm việc bao gồm tùy chọn người dùng, lịch sử tương tác và các hành vi đã học, cho phép cải thiện liên tục và cá nhân hóa theo thời gian. |
| **Insight**        | Giám sát và debug các agent đã triển khai bằng cách sử dụng nhật ký Runtime, nhật ký endpoint và các chỉ số tài nguyên (CPU/RAM).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |

GreenNode đang phát triển các module cho **kiểm soát và quản trị** Agent. Các cập nhật về các module mới này sẽ sớm được công bố.

## Cách Sử Dụng Các Module AgentBase

Mỗi module trong AgentBase có thể được truy cập thông qua bốn phương thức tương tác. Trong tài liệu của mỗi module, bạn sẽ tìm thấy hướng dẫn thiết lập cho từng phương thức — hãy điều hướng đến phương thức phù hợp với quy trình làm việc của bạn.

| Phương thức        | Mô tả                                                                                                                                                                                                                |
| ------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Portal**   | Giao diện quản lý trên web để cấu hình và quản lý các agent của bạn một cách trực quan, không cần viết code.                                                                                                                    |
| **Skills**   | Các tích hợp sẵn có mang khả năng AgentBase trực tiếp vào các công cụ phát triển hỗ trợ AI. Ví dụ, AgentBase Skills cho Claude Code cho phép bạn quản lý agent trực tiếp từ terminal bằng ngôn ngữ tự nhiên. |
| **SDK**      | Các thư viện chính thức để tương tác với AgentBase theo cách lập trình trong mã nguồn ứng dụng của bạn.                                                                                                                           |
| **REST API** | Các cuộc gọi HTTP trực tiếp đến các endpoint AgentBase, tương thích với bất kỳ ngôn ngữ hoặc công cụ nào.                                                                                                                                            |
