# Agent Base 1-Click (OpenClaw)

## 1. Agent Base 1-Click là gì?

**Agent Base 1-Click** tích hợp với **OpenClaw** cho phép triển khai AI Agent nhanh chóng chỉ với một thao tác, không yêu cầu kỹ năng lập trình. Phù hợp với business user muốn ứng dụng AI vào quy trình vận hành mà không cần đội ngũ kỹ thuật chuyên sâu.
