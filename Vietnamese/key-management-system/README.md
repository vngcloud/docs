# Key Management System

GreenNode KMS giúp người dùng quản lý Key một cách an toàn và hiệu quả. GreenNode hỗ trợ người dùng hai loại Key:

* GreenNode Managed Key: Key do GreenNode tạo tự động theo yêu cầu khi người dùng sử dụng các dịch vụ của GreenNode hoàn toàn miễn phí. Người dùng chỉ có thể xem thông tin chi tiết key.
* Customer Managed Key: Key do người dùng tạo hoặc import trên hệ thống KMS của GreenNode. Người cùng có thể thay đổi trạng thái key hoặc xóa key khi không còn sử dụng.   &#x20;
