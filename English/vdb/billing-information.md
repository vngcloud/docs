# Billing Information

The Billing Dashboard provides you with an overview of the DB cloud instance types you are currently using, along with important payment information and related actions. Here, you can:

* Renew instance usage period to avoid service interruption.
* Recover instance resources after expiration.
* Stop POC service and switch to regular usage.
* Enable/disable auto-renewal to manage payments more conveniently.

## Key Features

**DB Instance Type List:**

* Displays all DB cloud instance types you are currently using, including information about configuration, pricing, and remaining usage time.
* You can filter and sort the list by criteria such as instance name, instance type, and payment status.

**Renew:**

* Allows you to renew the usage period of one or more DB cloud instances.
* Select the instance to renew, choose the renewal period (e.g., 1 month, 3 months, 6 months, 1 year) and confirm payment.
* This feature helps you avoid service interruption due to expiration.

**Recover Resources:**

* If your DB cloud instance has expired and been deactivated, you can recover resources using this feature (within 7 days from expiration).
* Select the instance to recover and confirm payment.
* Please note that your data may be lost after a certain period from expiration, so recover resources as soon as possible.

**Auto-renew:**

* This feature allows you to automatically renew the usage period of DB cloud instances when they are due.
* Auto-renewal helps you save time and ensures uninterrupted service.

**Stop POC (Stop POC and switch to regular usage)**

* This feature allows you to switch a resource currently in POC to regular usage.
* We recommend that users proactively stop POC after completing all necessary testing before the POC expires, to avoid service interruption.

## **User Guide**

* **Step 1 - Access Billing Dashboard:** Log in to your account and access the Billing Dashboard here: [https://vdb.console.vngcloud.vn/billing](https://vdb.console.vngcloud.vn/billing)
* **Step 2 - Perform action:** Select Action on the instance you want to manage (refer to the image below)
  * Click the "Renew resource" button to renew the usage period or recover resources after expiration.
  * Click the "Stop POC" button to switch to regular usage.
  * Toggle the "Auto-renew" switch to manage this feature.
