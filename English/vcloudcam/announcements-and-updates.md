# Announcements & Updates

#### New Features:

* Azure AD Login for "Local Admin" and "Operator" accounts: Administrative accounts can now create and authorize subordinate users to log in via Username/Password or through Azure AD using email (Note: Mobile app is not yet available).
* Camera Recording Status: Local Admin accounts can now view a list of cameras that "have stopped recording".
* Organizational AI Face Recognition Access: Organization accounts can now access the AI Face Recognition features of sub-accounts within the same organization to view both live feeds and recognition history.
