# View Repository History

Access a Repository to view the history of actions performed on it, including the following information:

* **User Name:**\
  The actor who directly performed the action.
  * **Repository User:**\
    An entity that directly interacts with Images in the Repository.
  * **Default User:**\
    Each GreenNode Account is associated with one Default User, which is used to create Repositories, create Repository Users, and perform other management operations.
  * **Anonymous:**\
    Used for actions performed on a Public Repository without authentication through a Repository User.
* **Resource Name:**\
  The name of the affected resource, which can be a Repository Name or Image Name.
* **Resource Type:**\
  The affected resource type, including Repository, Image, or Image Artifact.
* **Action Name:**\
  The name of the action performed on the resource.
* **Action Time:**\
  The time when the action was performed.
