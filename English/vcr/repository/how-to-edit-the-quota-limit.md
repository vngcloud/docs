# How to Edit the Quota Limit

Refer to the following instructions to adjust the quota limit of your Repository.

#### Steps to Adjust the Repository Quota Limit

On the Container Registry homepage, click the **"Repository"** menu on the right-hand side to access the Repository list page. In the **"Action"** column, click the **three-dot icon** on the Repository you want to modify, then select **"Edit Quota Limit"** to begin adjusting the quota limit.

An **"Edit Quota Limit"** popup will appear, allowing you to modify the following:

* **Quota Limit:**\
  The current limit is displayed by default. Users can enter a new quota limit value based on actual usage requirements.\
  Note: The system will not allow the quota limit to be reduced below the Repository’s current storage usage.

Click **"Save"** to complete the adjustment process.
