# vCR

GreenNode is pleased to introduce a new service within the GreenNode ecosystem — Container Registry (vCR). To enhance the user experience, Container Registry provides a user-friendly web portal and seamless integration with other internal cloud services, enabling you to easily manage and deploy your containerized applications.

### Key Features

#### User-Friendly Web Portal

A simple and intuitive interface for easy navigation and management of container images.

#### Seamless Integration

Smooth integration with other internal cloud services, allowing easy deployment of container images.

#### IAM Permissions

Easily configure permissions and access control using GreenNode IAM (Identity and Access Management), ensuring secure and controlled access to users’ container images.

### Benefits of Container Registry

#### Improved Efficiency

Simplifies the process of managing and deploying container images, reducing operational complexity.

#### Streamlined Workflow

Seamlessly integrates container deployment into your existing cloud infrastructure.

#### Enhanced Security

Effectively control access to your container images through IAM permissions.

### Getting Started

#### Access the Web Portal

Simply log in to your GreenNode account and navigate to the Container Registry section to get started at:\
[https://vcr.console.vngcloud.vn/](https://vcr.console.vngcloud.vn/)

#### Explore the Service

Explore Container Registry by creating Repositories to store container images, then create Repository Users to Push/Pull images and manage them within the Repository.

#### Configure Permissions

Set up IAM-based permissions to efficiently manage access control and security.
