# Agent Base 1-Click (OpenClaw)

## 1. What is Agent Base 1-Click?

**Agent Base 1-Click** integrates with **OpenClaw** to allow you to deploy an AI Agent with a single click — no coding required. Ideal for users who want to apply AI to their workflows without a dedicated technical team.
