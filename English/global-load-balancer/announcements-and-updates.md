#### November 12, 2024

### Initial Release: Network Global Load Balancer – Expanding Global Reach

GreenNode is pleased to announce the launch of the first release of the Global Load Balancer (GLB), marking a significant milestone in optimizing the performance and availability of your applications across multiple geographic regions.

1. **From Load Balancer to Global Load Balancer: What’s Different?**

While traditional Load Balancers focus solely on distributing traffic within a single VPC network, Global Load Balancer introduces a completely new approach:

* **Multi-Region Coverage:** GLB enables you to manage and distribute traffic across servers located in multiple geographic regions, allowing your applications to serve users worldwide.
* **Performance Optimization:** By routing requests to the server closest to the user, GLB minimizes latency and improves response speed.
* **High Availability:** GLB distributes risk across regions, ensuring service continuity even if one region experiences failures.

2. **Key Features**

* Support for Network-type GLB: Efficient TCP traffic load balancing.
* **Flexible Configuration:** Create and manage Global Pools, configure load balancing algorithms, and set up health checks.
* **Geolocation Routing:** Intelligent routing based on the user’s geographic location.
* **User-Friendly Interface:** Provides an intuitive interface for managing and configuring GLB.

3. **Benefits**

* Improved Performanc&#x65;**:** Reduce response times for users, especially for high-traffic applications.
* High Reliabilit&#x79;**:** Distribute risks and maintain uninterrupted services even when a region encounters issues.
* Scalability: Easily add or remove servers from the pool without affecting system operations.
