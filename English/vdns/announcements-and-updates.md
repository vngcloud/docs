#### December 16, 2024

GreenNode introduces vDNS, a powerful DNS service designed to optimize performance, enhance security, and ensure the stability of your online infrastructure. vDNS provides fast and reliable domain name resolution, ensuring a smooth access experience for users.

### Supported DNS Record Types

vDNS fully supports common DNS record types, including:

* **A:** Points a domain name to an IPv4 address.
* **CNAME:** Creates an alias for another domain name.
* **MX:** Specifies the mail server responsible for receiving emails for the domain.
* **SRV:** Specifies the location of specific services, such as VoIP services.
* **PTR:** Maps an IP address back to a domain name (reverse DNS).
* **TXT:** Stores arbitrary text, commonly used for verification and other purposes.

### Routing Policies

vDNS provides flexible routing policies to optimize performance and user experience:

* **Simple Routing:** Simple routing that returns a set of resources in order.
* **Geolocation Routing:** Routes traffic based on the user’s geographic location, enabling traffic distribution to the nearest servers.
* **Weighted Routing (with Sticky Session):** Distributes traffic based on weights assigned to each resource. The Sticky Session feature ensures users are routed to the same resource throughout their session.
